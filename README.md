# WonderText

WonderText is a powerful and user-friendly text manipulation and analysis tool. It provides a variety of utilities to transform, analyze, and enhance your text, including text case conversion, Morse code translation, speech synthesis, and more. Built with modern web technologies, WonderText is designed to be fast, responsive, and easy to use.

---

## ğŸš€ Features

- **Text Transformation**:
  - Convert text to uppercase or lowercase.
  - Remove extra spaces.
  - Clear text with a single click.

- **Clipboard Utilities**:
  - Copy text to the clipboard.
  - Paste text from the clipboard.

- **Speech Synthesis**:
  - Read text aloud using the browser's speech synthesis API.
  - Stop speech at any time.

- **Morse Code Conversion**:
  - Convert text to Morse code.
  - Decode Morse code back to text.

- **Local Storage**:
  - Save text to local storage.
  - Load saved text from local storage.
  - Clear saved text from local storage.

- **Markdown Support**:
  - Preview text with Markdown formatting.

- **Statistics**:
  - Word count.
  - Character count.
  - Estimated reading time.

- **Responsive Design**:
  - Fully responsive and works seamlessly on all devices.

## ğŸ› ï¸ Installation and Setup

Follow these steps to set up WonderText locally:

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/HarshYadav152/WonderText.git
   cd WonderText
   cd WonderText
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
